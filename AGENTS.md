# AGENTS.md

このファイルは、AIエージェント（Claude Code / GitHub Copilot Workspace / その他のLLMベースアシスタント）が本リポジトリで作業する際に**必ず守るべきルール**を定めたものです。
人間の開発者も、本サイトの方針として参考にしてください。

## サイトの位置づけ

- 本サイトは、**浄土真宗本願寺派 大船山 金剛寺**（神奈川県鎌倉市玉縄）の **公式サイト** である
- 静的な案内サイトであり、**予約システムではない**
- 主な目的は、寺院の情報提供と、ご相談を電話へつなぐことである

## 絶対に守ること

### 宗教的内容について

- **仕様にない宗教サービスを勝手に追加しない**
- 特に以下は、本サイトに**勝手に追加してはならない**：
  - 祈祷（きとう）
  - 厄除け
  - 水子供養
  - 御朱印
  - その他、浄土真宗本願寺派の方針に沿わない可能性のあるサービス
- 他宗派的に見える装飾・図像・文言も避ける

### 「合葬墓」「お骨預かり」について

これらは、今後 金剛寺側で正式な文言を確定する予定のセクションである。

- **以下の内容を推測で書かない**：
  - 金額・費用
  - 利用条件
  - 宗派の可否（「宗派不問」「どなたでも利用可能」など）
  - 永代供養の可否
  - 管理費・後継者の要否
  - 生前予約の可否
  - 改葬の可否
  - 納骨方法・期間
- 不確かな文言は `{/* TODO[content]: ... */}` として残す
- 「詳細はお電話にてお問い合わせください」程度に留める

### 駐車場について

- 駐車場は **3台ほど** あること、台数に限りがあることを必ず明記する
- 確定文言（変更しないこと）：
  > 駐車場は3台ほどございます。台数に限りがございますので、お車でお越しの際は、必要に応じて事前にお問い合わせください。
- 「広い駐車場完備」「何台でも駐車可能」など、断定的・営業的すぎる表現は避ける

## デザイン・実装ルール

### トーン

- 上品、静か、清潔感がある、現代的
- 高齢者にも読みやすい（十分なフォントサイズ・行間・コントラスト）
- お寺らしい安心感がある

### カラー

`tailwind.config.mjs` 定義のトークンのみ使用する。
新色を追加する場合は、トークンに追加してから使うこと。

- 基調：白（paper）、生成り（paperWarm）、墨色（ink, inkSoft）
- アクセント：淡い藤色（fuji, fujiSoft）、薄い金色（gold, goldSoft）
- 強い黒、原色、金ピカ、過度な装飾は避ける

### フォント

- 見出し：`font-serif`（Noto Serif JP）
- 本文：`font-sans`（Noto Sans JP）

### アニメーション

- 控えめにする（派手な動きは禁止）
- `prefers-reduced-motion: reduce` を**必ず**尊重する
- 外部アニメーションライブラリ（GSAP、Framer Motion 等）は追加しない
- スクロール表示は `FadeIn.astro` を使用
- パララックスは `Parallax.astro` のみ使用（モバイル無効化済み）

### 画像

- 画像素材は `src/assets/placeholders/` 配下に置き、Astro `<Image />` で読み込む
- ファイル名を維持して差し替えると、コード変更不要で反映される
- 仮画像は、中立的なSVGグラデーション。最終的な写真は金剛寺側で用意される
- 画像差し替えのTODOは `{/* TODO[image]: ... */}` で示す

### コンテンツ

- 文言が不確かな場合は、必ず `{/* TODO[content]: ... */}` として残す
- 「絶対」「必ず」「100%」「最高」など断定的・営業的すぎる表現を避ける
- 仏教用語は浄土真宗本願寺派の作法に沿った表現を使う（不明な場合はユーザーに確認）

## 寺院情報の編集

寺院の確定情報（住所・電話・宗派・アクセスなど）は `src/data/site.ts` を **単一の真実の情報源** として編集する。各ページに直接書き込まない。

## アクセシビリティ

- すべてのインタラクティブ要素にキーボードフォーカスが当たること
- ボタン・リンクに適切な `aria-label` を付ける
- `lang="ja"` を維持
- 装飾的画像には `alt=""` `role="presentation"` を付ける
- 電話番号は `tel:` リンクで提供する

## SEO

- 各ページに固有の `<title>` と `<meta name="description">` を設定する
- OGP画像は 1200×630 を維持
- 構造化データ（JSON-LD `BuddhistTemple`）は `BaseLayout.astro` で管理

## コミットメッセージ

例：

- `feat(top): 法話会セクションを追加`
- `content(gassobo): 合葬墓の正式文言に差し替え`
- `image(hero): 本堂外観写真に差し替え`
- `fix(access): 駐車場文言を確定文に統一`
